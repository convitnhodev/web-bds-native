# deeincom

Mọi người sửa UI nhớ build css: `cd node && npm run build`. Nếu ko sẽ thiếu style :)).


# requirement

golang 1.16 hoặc mới hơn
postgresql 12 hoặc mới hơn

# chạy

```
go run cmd/deein/main.com web
```

# Sử dụng docker để run app thì:
+ Install docker

```
cd ./project
docker compose up
```
